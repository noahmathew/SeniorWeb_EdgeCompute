Multi Modal Edge AI System for Anomaly Detection in Pharmaceutical Manufacturing

About

This project implements a multi-modal edge AI system for real-time anomaly detection in pharmaceutical manufacturing. Our solution uses a fine-tuned YOLO classifier to detect cracks and verify proper vial capping, enabling quality assurance at the edge with minimal latency.
